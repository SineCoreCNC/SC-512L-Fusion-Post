# SC-512L Fusion 360 Post Processor

Official Fusion 360 post processor downloads for the **SineCore SC-512L CNC Lathe**.

This repository is used to distribute the latest Fusion 360 post processor files for the SC-512L.

## Download Latest Post

Download the latest version here:

https://github.com/SineCoreCNC/SC-512L-Fusion-Post/releases/latest

## Installation

1. Download the latest `.cps` file from the Releases page.
2. In Fusion 360, open your post library.
3. Import the downloaded post processor.
4. Select it when posting programs for the SC-512L.

## Notes

- Use the latest release unless instructed otherwise.
- Release notes are included with each update.
